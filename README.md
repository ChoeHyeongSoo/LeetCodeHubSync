# 🚀 LeetCode Auto-Sync (BaekjoonHub Style)

리트코드(LeetCode)에서 푼 문제를 내 깃허브 레포지토리에 **백준허브 스타일**로 자동 저장해주는 도구입니다. 매일 정해진 시간에 봇이 알아서 문제를 긁어오며, 문제별로 **난이도 분류, 런타임/메모리 측정, 개별 커밋**까지 완벽하게 지원합니다.

## ✨ 주요 기능
- **백준허브 스타일 구조**: `LeetCode / 난이도 / [번호] 문제제목 / README.md & Solution.java`
- **상세 리드미 생성**: 성능 요약, 알고리즘 분류, 문제 본문 포함
- **깔끔한 커밋 메시지**: 문제 번호, 제목, 언어, 성능 수치를 포함한 개별 커밋
- **완전 자동화**: 매일 정해진 시간(한국 시간 새벽)에 자동 실행

---

## 🛠 세팅 방법 (3분 소요)

친구들은 아래 순서대로만 따라 하면 바로 본인의 잔디를 채울 수 있습니다.

### 1. 레포지토리 포크 (Fork)
- 우측 상단의 **[Fork]** 버튼을 눌러 본인의 계정으로 이 레포지토리를 가져옵니다.

### 2. 리트코드 세션(Cookie) 가져오기
봇이 본인의 계정에 접속할 수 있는 '열쇠'가 필요합니다.
1. 크롬(Chrome) 브라우저에서 [LeetCode](https://leetcode.com/)에 로그인합니다.
2. `F12`를 눌러 개발자 도구를 엽니다.
3. 상단 탭에서 **[Application]** (또는 애플리케이션)을 클릭합니다.
4. 왼쪽 메뉴에서 **[Cookies]** ➔ `https://leetcode.com`을 선택합니다.
5. 목록에서 **`LEETCODE_SESSION`**을 찾아 `Value` 값을 복사합니다. (매우 기니까 끝까지 복사하세요!)

### 3. 깃허브 시크릿(Secrets) 등록
1. 포크한 본인의 레포지토리 **[Settings]** ➔ **[Secrets and variables]** ➔ **[Actions]**로 이동합니다.
2. **[New repository secret]** 버튼을 누릅니다.
3. Name: `LEETCODE_SESSION`
4. Value: 아까 복사한 세션 값을 붙여넣습니다.

### 4. 봇 권한 및 아이디 설정
1. **권한 허용**: 레포지토리 **[Settings]** ➔ **[Actions]** ➔ **[General]** 맨 아래에서 `Workflow permissions`를 **`Read and write permissions`**로 바꾸고 **[Save]**를 누릅니다.
2. **아이디 수정**: `scripts/sync_leetcode.js` 파일을 열고 19번 라인의 `username: ""` 부분을 **본인의 리트코드 아이디**로 수정합니다.

### 5. 실행 테스트
1. 상단 **[Actions]** 탭으로 이동합니다.
2. 왼쪽에서 **[LeetCode Sync Action]**을 선택합니다.
3. 우측의 **[Run workflow]** 버튼을 눌러 봇을 수동으로 깨웁니다.
4. 잠시 후 초록색 체크(`✔`)가 뜨면 성공! 본인의 레포지토리에 `LeetCode` 폴더가 생겼는지 확인하세요.

---

## ⚠️ 주의사항
- 리트코드 사이트에서 **[로그아웃]**을 하면 세션 값이 무효화됩니다. 로그아웃하지 않도록 주의하세요.
- 만약 봇이 작동하지 않는다면 세션 값이 만료된 것이니, 2번 단계부터 다시 해서 시크릿을 업데이트해주면 됩니다.
